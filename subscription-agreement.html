  • "用户"或"您"指订阅会员服务的个人或实体

会员可享受以下权益：

  • 会员权益仅限订阅账户使用，不可转让

3. 订阅规则

3.1 订阅方式

会员服务通过Apple In-App Purchase进行订阅，支付将通过您的Apple ID账户完成。

3.2 自动续订

重要：自动续订说明
  • 订阅将自动续订，除非在当前订阅期结束前至少24小时取消
  • 续订费用将在当期订阅期结束前24小时内从您的Apple ID账户扣除
  • 您可以在Apple ID账户设置中管理订阅和关闭自动续订
  • 购买后，订阅立即生效

3.3 订阅价格

订阅价格以应用内显示为准，我们保留调整价格的权利。价格调整不会影响您当前的订阅周期，但会在下一个订阅周期生效。

3.4 订阅限制

  • 您必须年满18周岁或已获得监护人同意方可订阅
  • 订阅前请确保您的Apple ID账户有足够的余额或已绑定有效的支付方式
  • 同一Apple ID在同一时间只能有一个有效订阅

3.5 订阅确认

完成订阅后，会员权益将立即生效。如因网络或其他原因导致延迟生效，我们将在24小时内处理。

4. 取消订阅

4.1 取消方式

您可以通过以下方式取消订阅：

  • 在iOS设备上：设置 → Apple ID → 订阅 → 美颜换换 → 取消订阅
  • 在Mac上：App Store → 账户 → 订阅 → 美颜换换 → 取消订阅
  • 取消后，当前订阅周期结束后将不再续订

4.2 取消生效时间

取消订阅后，您仍可享受会员权益至当前订阅周期结束。订阅周期结束后，会员权益将自动终止。

4.3 退款政策

根据Apple的退款政策，已订阅的费用通常不予退款。但在以下情况下，您可以申请退款：

  • 因技术故障导致重复扣款
  • 因系统错误导致订阅未生效
  • 未成年人未经监护人同意进行的订阅

如需申请退款，请通过邮箱 support@faceglow.app 联系我们，或通过Apple的退款渠道申请。
